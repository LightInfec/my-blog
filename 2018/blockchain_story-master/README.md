
##  我的区块链故事


终于有时间来写一些东西了,把半年来的经历和所了解到的知识整理一下,但是限于篇幅,还是有部分的东西没有写进去.希望它能给你一些启发.


文章全篇从17 年11 月币圈泡沫期开始,包含内容有区块链技术,区块链产业相关,区块链安全,区块链金融等相关内容.旨在于让读者理性看待区块链,不要被造富神话迷惑.
